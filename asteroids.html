Unreal Engine 4/5 | 2023

ROLE: Sole Developer

This project is a Demo game of an asteroid shooter. Asteroids will be procedurally generated around the spaceship. The ship can activate boost mode, a bullet time mode once the bullet time bar is filled. Crystals will be spawned from destroyed asteroids. The project has a fully functional UI system.
